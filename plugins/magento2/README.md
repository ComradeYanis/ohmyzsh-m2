# Magento 2 plugin

The plugin adds autocomplete commands and subcommands for [magento2](https://devdocs.magento.com/).

To use it, add `m2` to the plugins array of your zshrc file:

```zsh
plugins=(... m2)
```

## Aliases

| Alias | Command       |
|-------|---------------|
| m2    | `bin/magento` |
